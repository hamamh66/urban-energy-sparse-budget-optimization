# Urban Energy Coordination with Sparse-Budget Optimization

This repository provides a reproducible implementation of a temporal
sparse-budget optimization framework for urban energy coordination. It
integrates forecasting, threshold-aware optimization, and interpretable
intervention strategies for realistic smart-grid operation.

## Features

-   Peak load reduction and excess minimization
-   Threshold-aware violation analysis
-   Sparse-budget intervention control
-   Storage and load-shifting modeling
-   Reproducible notebook and outputs

## Structure

-   notebooks/: Main experiments
-   figures/: Generated figures
-   tables/: Result tables
-   outputs/: Summary results

## Reproducibility

All results are generated via Google Colab with outputs saved to Google
Drive.
